Assignment 2 - GRLMAT001

Program can be compiled using "make release", and run using command line arguments eg. ./volimage brain_mri_raws/MRI.data -d 40 45 diffmap.raw

Files:
    driver.cpp - contains ONLY the Main() method
    volimage_h.h - header file for declaring methods and variables
    volimage.cpp - contains all methods and calculations to be used in the program eg. diffmap, extract, etc
    Makefile - contains commands to build the Program
